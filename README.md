# Сокет-сервер с клиентом и Redis Pub/Sub

## 📌 Описание
Реализация асинхронного TCP-сервера с:
- Поддержкой множества клиентов
- Интеграцией Redis Pub/Sub
- Механизмом подписок
- Логированием операций

## 🚀 Быстрый старт

### Требования
- .NET 8.0+
- Redis 5.0+
- 5000 порт (или любой свободный)
